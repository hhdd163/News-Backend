# FastAPI 新闻项目 - 学习指南

## 🎯 项目简介

这是一个**完整的头条新闻后端系统**，使用 FastAPI 框架构建。项目实现了用户管理、新闻浏览、收藏功能等核心业务，采用现代化的异步编程模型和分层架构设计。

### 第二阶段：掌握数据模型（2-3天）

#### 1. 学习 SQLAlchemy ORM
**文件**: `models/*.py`

**核心概念**:
- 声明式基类 `DeclarativeBase`
- 类型注解 `Mapped[T]`
- 字段映射 `mapped_column()`
- 关系定义 `ForeignKey()`
- 索引优化 `Index()`

**示例**:
```python
class News(Base):
    __tablename__ = "news"
    
    id: Mapped[int] = mapped_column(Integer, primary_key=True)
    title: Mapped[str] = mapped_column(String(255), nullable=False)
    category_id: Mapped[int] = mapped_column(Integer, ForeignKey('news_category.id'))
```

#### 2. 理解表关系
- **一对多**: Category → News（一个分类有多条新闻）
- **多对多**: User ↔ News（通过 Favorite 表关联）

### 第三阶段：学习数据操作（3-4天）

#### 1. 掌握 CRUD 操作
**文件**: `crud/*.py`

**核心技能**:
- 异步查询 `select()`
- 条件过滤 `where()`
- 分页处理 `offset().limit()`
- 数据更新 `update()`
- 数据删除 `delete()`
- 聚合查询 `func.count()`

**示例**:
```python
# 查询列表
stmt = select(News).where(News.category_id == category_id)
result = await db.execute(stmt)
news_list = result.scalars().all()

# 分页查询
stmt = select(News).offset(skip).limit(page_size)

# 更新数据
stmt = update(News).where(News.id == news_id).values(views=News.views + 1)
```

#### 2. 理解联表查询
```python
# JOIN 查询
query = (
    select(News, Favorite.created_at.label("favorite_time"))
    .join(Favorite, News.id == Favorite.news_id)
    .where(Favorite.user_id == user_id)
)
```

## 💡 最佳实践

### 1. 代码组织原则

✅ **分层清晰**: routers → crud → models  
✅ **单一职责**: 每个函数只做一件事  
✅ **命名规范**: 见名知意，使用英文  
✅ **注释完善**: 关键逻辑添加注释  

### 2. 数据库优化技巧

✅ **索引优化**: 为常用查询字段建立索引  
✅ **分页查询**: 使用 offset/limit 避免全表扫描  
✅ **联表查询**: 减少数据库访问次数  
✅ **连接池**: 合理配置 pool_size 和 max_overflow  

### 3. 缓存使用策略

✅ **热点数据**: 分类、列表等频繁读取的数据  
✅ **分级过期**: 不同数据设置不同过期时间  
✅ **缓存雪崩**: 避免大量 key 同时过期  
✅ **缓存穿透**: 查询不存在的数据也要缓存  

### 4. 安全注意事项

✅ **密码加密**: 使用 bcrypt，不存储明文  
✅ **Token 验证**: 所有敏感接口都需要认证  
✅ **SQL 注入**: 使用 ORM，不拼接 SQL  
✅ **CORS 配置**: 生产环境限制允许的域名

## 🛠️ 常见问题解答

### Q1: 如何处理异步数据库会话？

**A**: 使用 `async with` 和 `yield` 实现生命周期管理：

```python
async def get_db():
    async with AsyncSessionLocal() as session:
        try:
            yield session
            await session.commit()
        except Exception:
            await session.rollback()
            raise
        finally:
            await session.close()
```

### Q2: 如何实现分页？

**A**: 使用 offset 和 limit：

```python
skip = (page - 1) * page_size
stmt = select(News).offset(skip).limit(page_size)
```

### Q3: 如何验证用户身份？

**A**: 使用依赖注入自动验证 Token：

```python
@router.get("/protected")
async def protected_route(user: User = Depends(get_current_user)):
    # 只有携带有效 Token 的请求才能访问
    ...
```

### Q4: 如何处理跨域请求？

**A**: 配置 CORS 中间件：

```python
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

### Q5: 如何统一响应格式？

**A**: 使用工具函数：

```python
return success_response(message="成功", data={...})
```
